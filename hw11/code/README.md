WheatonConnections
===
[Wheaton Course Connections](http://wheatonconnections.me/)

This website uses [uikit](http://getuikit.com/index.html) as the CSS front-end frameworks, and aims to help Wheaton College (MA) students to browse and choose [connection courses](http://wheatoncollege.edu/academics/the-wheaton-curriculum/connections/connections-list/) more easily. The link of website can be found here: http://wheatonconnections.me/
